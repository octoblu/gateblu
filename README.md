gateblu
==============

Gateblu will automatically register itself with meshblu.octoblu.com on startup.

```
npm start
```

Sample `meshblu.json`

```
{
  "server": "meshblu.octoblu.com",
  "port": "80",
  "devicePath": "devices",
  "tmpPath": "tmp",
  "uuid": "c4110691-69fd-11e4-b75c-63db18512b02",
  "token": "0kmpejq8ul434n29pbti61d2q4vg3nmi"
}
```
